# mapGuessing 根据地名猜地理位置

这个小玩具某种程度上算是我一直想要实现想要拥有的，用于增强自己对某些城市所在地理位置的印象/感知的，寓教于乐的小玩意。

玩法就是根据上方说明的国家-城市，点选图上的marker。

左上角有帮助以及筛选的按钮，可以根据需求使用。

> 关于帮助按钮，其中有反馈链接，你可以填写告诉我你想要添加的城市，但是我不一定真给添上，我个人会倾向于一些历史意义比较大的城市，不过考虑到雨露均沾，也许我会采纳你的建议也说不定。
>
> 关于子区域这块，我没有按照国家来判定，比如我国甘肃新疆的一些城市我选择归到中亚，也许改叫内亚比较好，总之这里我想要尽量让这个子区域划分更有趣一点，而不要被国家定义而左右。

右上角有cheat mode的按钮可以来看都有哪些城市。

以及右侧还有一个layer button这个可以更换底图、切换是否加载国界线、切换到困难模式（更少容错）。

点击marker之后会弹出一个ai总结的对应城市的wiki内容的pop-up，也算是一种长知识的方式吧。

总之，enjoy。
